Git Command

git init

Used to initialize folder to work with git

git status

Used to check status of working directory

git add

Add working directory to statging directory

git commit -m "message"

Update change in local/remote directory from statging directory

git remote add origin "git-repo-url"

Connect central(github)/remote repository to local directory

git push -u origin main

Add user id & password to push local change to remote(github)
